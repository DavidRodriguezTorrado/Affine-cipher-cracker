# Affine-cipher-cracker
This repository allows you to decode any text encrypted with Affine Algorithm. You can also get the keys (A and B) once you recognize a text.

# How to use
Substitute the variable "Text_to_decode" by the text you want to decode.
When running affine.py all rotational posibilities will be printed.

# Make sure before running
Note that English dictionary is used. This script should be modify to decode other language dictionary.
